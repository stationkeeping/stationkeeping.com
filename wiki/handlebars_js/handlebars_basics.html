Basic

Example of a formatting:

```

{{title}}

```

  1. Make the template available on the page using a script tag:

```

{{title}}

</script> ```

  1. Compile it to JavaScript using:

var source = $("#title-template").html(); var template = Handlebars.compile(source);

  1. Execute and populate using:

var context = {title: "Example Title"} var html = template(context);

Helpers

Register a helper:

Handlebars.registerHelper('capitalize', function(title) { return title.capitalize(); });

Then Use it:

```

{{capitalize title}}

```
