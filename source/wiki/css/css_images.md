---
title: "CSS Images"
---

Ensure an img maintains it's proportions:

```
.image-wrapper img {
  max-width:100%;
  height:auto;
}
```

Image replacement for headers (HTML5BP Version):

h1
{
  border: 0;
  font: 0/0 a;
  text-shadow: none;
  color: transparent;
  background: url(example.png);
  width: 300px;
  height: 75px;
}

Run any images through [ImageOptim](http://imageoptim.com/) to get maximum compression.
